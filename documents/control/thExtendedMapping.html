thExtendedMapping

The thExtendedMapping controls the use of extended mapping in Thymol URI processing.

Extended mapping is the process by which the test for a match between a mapping key value and an URI is found true if the initial substring of the URI matches the key.

The default value for this parameter is "false". The default value is used if it isn't overridden by setting thExtendedMapping.
