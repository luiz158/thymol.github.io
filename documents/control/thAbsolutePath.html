thAbsolutePath

The thAbsolutePath may be used to override the path value used in template resolution.

The override will only be enforced if the thUseAbsolutePath parameter is in effect (set to "true").

The path value will be prefixed by any thProtocol value that is in effect.

The relative fragment from the template is appended to the resultant URI before resolution takes place.
