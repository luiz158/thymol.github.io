thymol files

The Thymol 2.0 distribution contains a bootstrap loader and configuration module in "thymol.js".

The Thymol loader will load one of 2 distinct variants of the Thymol library these are: "thymol-full.js" and "thymol-lite.js".

The first, "thymol-full.js" provides the full range of Thymol 2.0 functionality as described in the "what's new in 2.0.0" section of features page, as well as that of previous versions of Thymol.

The other file "thymol-lite.js", provides all Thymol 2.0 functionality apart from support for expression utility objects.

The reduced functionality version has been included as a direct replacement for earlier (pre-2.0) versions of Thymol for users who don't need to use expression utility objects, and who may have concerns over javascript file load times.

Along with these files, the distribution also contains "minimised" versions of each of the above: "thymol.min.js", "thymol-full.min.js" and "thymol-lite.min.js".
