thMessagesBaseName

Thymol uses the thMessagesBaseName value to set-up potential names for messages properties files and uses those potential values as the initial targets of the searching process.

The default value for thMessagesBaseName is "Messages", so that by default, the search attempts to find files named "Messages.properties" or "Messages(suffix).properties" in the directory specified by thMessagePath.
