dialects

Thymol 2.0 extensions may optionally be grouped together into a logical set sharing the same xml namespace prefix. You do this by invoking the thymol.addDialect function.

thymol.addDialect = function(dialectSpec)

The single mandatory argument "dialectSpec" names a specification object that should have the following structure:

        {
            prefix : prefixNm,
            attributeProcessors : [ 
                {
                    name : attr1Nm,
                    processor : attr1ProcessorFnNn,
                    precedence : attr1Prec
                },
                {
                    name : attr2Nm,
                    processor : attr2ProcessorFnNn,
                    precedence : attr2Prec
                }, 
                ...
            ],
            elementProcessors : [
                {
                    name : elem1Nm,
                    processor : elem1ProcessorFnNn
                },
                {
                    name : elem2Nm,
                    processor : elem2ProcessorFnNn
                }, 
                ...
            ],
            objects : [
                obj1ExprObj,
                obj2ExprObj,
                {
                    name : obj1Nm,
                    object : obj1ProcessorFnNn
                },
                {
                    name : obj2Nm,
                    object : obj2ProcessorFnNn
                },
                ...
                ...
            ]
        }

Where:

  • Any of the properties prefix, attributeProcessors, elementProcessors or objects may be omitted.
  • prefixNm is the (xml namespace) prefix used to identify the dialect components.
  • attrNProcessorFnNn is a reference to the Nth atrribute processor function.
  • attrNPrec is the precedence value to assign the Nth attribute (integer low = process early, high = process late, default 20000).
  • objNExprObj is a reference to the Nth configureModule loadable expression object definition.
  • objNNm is the name of the Nth plain javascript expression object (should start with a "#" character).
  • objNProcessorFnNn is a reference to the Nth Nth plain javascript expression object function.

Members of the attributeProcessors array property are name, function, precedence triples.

Members of the elementProcessors array property are name, function pairs.

The prefix value is shared by the attributeProcessors and elementProcessors.

Members of the objects array property may be either functions as defined on the expression objects page or plain javascript objects that expose functions to be called as expression object functions. In this case, the object field specifies the expression object function reference and the name field gives the expression object name.
