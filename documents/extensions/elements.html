elements

Thymol custom elements allow you to extend Thymol in a way that's similar to Thymeleaf's custom dialects.

Custom elements are defined by invoking the "thymol.configureElementProcessor" function.

thymol.configureElementProcessor = function(prefix, suffix, elementProcessorFunction)

  • prefix - specifies the "namespace" prefix value that introduces the element definition in the template. This is normally a short string, may be null to signify no prefix.

  • suffix - the name of the element being defined.

  • elementProcessorFunction - the definition of a callback function that is executed when the element is encountered during template processing.

    • elementProcessorFunction = function(element)

      • element - the DOM element instance to be processed.

The body of the elementProcessorFunction performs the intent of the custom element and may do what you will.

The elementProcessorFunction should return a boolean result value that is "true" if the function changed the DOM in any way and "false" if not.
