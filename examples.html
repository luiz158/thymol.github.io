examples

There are a number of new examples of Thymol usage in the 2.0 distribution download file (from sourceforge).

You can see some of these examples in action by trying the following links:

  • sessions the persistent session examples in the examples/templates/thymol/2.0/sessions directory.

  • say-hello the extending Thymeleaf in 5 minutes example (Thymol style) in the examples/templates/thymol/2.0/say-hello directory.

  • stsm example the extending Thymol to implement some Thymeleaf + Spring examples in the examples/templates/v21/stsm directory.

  • with-dialect a Thymol version of antibrumm's thymeleaf-extras-with-dialect in the examples/templates/v21/ecosystem/with-dialect directory.

  • extrathyme a Thymol version of the extrathyme custom dialect example in the examples/templates/v21/extrathyme/ directory.

  • this site the site you're currently browsing is built with Thymol 2.0. For full details go here.
