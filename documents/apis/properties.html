properties

The following objects provide JavaScript access to the well-known contexts:

  • thymol.applicationContext
  • thymol.sessionContext
  • thymol.requestContext

Each of these contexts exports a useful set of functions:

  • createVariable = function(name, value)

    Creates a variable that may be used in Thymol expressions.

  • getAttribute = function(name)

    Returns the named value (a generic object, may or may not be a variable).

  • addAttribute = function(name, value)

    Adds a generic object with a given name.
