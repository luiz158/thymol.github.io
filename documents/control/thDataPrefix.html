thDataPrefix

The thDataPrefix parameter sets the prefix used to identify html5 data-style attributes for processing by Thymol.

The default value for this parameter is "data".

For a given Thymol attribute, the actual value to be tested against when seeking html5-style instances of this attribute will be: 

    thDataPrefix + "-" + thPrefix + "-" + (attribute name)

For example, for the th:text attribute and default values for the control parameters, the corresponding html5-style attribute is: 

    data-th-text

The default value may be set to a different value using the thDefaultDataPrefix variable (see configuration section).
