thResourcePath

The thResourcePath control parameter defines a value that will be used to prefix the (relative) path to resources when generating Standard URL values.

For fully qualified URL values, the thResourcePath value is inserted just before the relative portion taken from the template (after the protocol, domain and path components).

For relative URL values, the thResourcePath value is similarly inserted at the head of the URI.

(See also thUseFullURLPath section).
