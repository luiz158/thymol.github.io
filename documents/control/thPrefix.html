thPrefix

The thPrefix parameter sets the prefix used to identify attributes for processing by Thymol.

The default value for this parameter is "th". The default value is used if it isn't overridden by setting thPrefix and cannot be deduced from an appropriate xmlns declaration in the html element.

The default value may be set to a different value using the thDefaultPrefix variable (see).
